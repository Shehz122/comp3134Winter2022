What is the definition of server port?
A server port number is a way to know a particular process to which an internert or other messages is to be forwarded when it arrives at the server.

What is server port range?
The range of server port is from 0 to 65535

How many bits does number above represent?
A port number uses 16 bits and so can therefore have a value from 0 to 65535 decimal.
